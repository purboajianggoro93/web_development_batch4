  1. HTML (Hypertext Markup Language)

    • Pengertian HTML

      HTML adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. Ini dapat dibantu oleh teknologi seperti Cascading Style Sheets (CSS) dan bahasa skrip seperti JavaScript dan VBScript.

    • Fungsi HTML

      Membuat struktur pada sebuah website.

    • Struktur HTML

      1. Head

        Berisikan meta data serta title.

      2. Body

        Berisikan konten yang nantinya akan tampil pada halaman browser.

    • Macam-macam Tag

      No. Nama Tag Keterangan
      1 Meta Tag ini terletak pada bagian head HTML
      2 Link Tag ini terletak pada bagian head HTML dan memiliki fungsi memanggil file.
      3 Style Tag ini terletak pada bagian head HTML dan memiliki fungsi menginputkan CSS.
      4 Heading Tag ini memiliki macam 6, yang masing-masing memiliki peran judul dan sub judul pada konten.
      5 Bold Tag ini memiliki fungsi membuat tampilan huruf terlihat tebal.
      6 Italic Tag ini memiliki fungsi membuat tampilan huruf terlihat miring
      7 Strong Tag ini memiliki fungsi yang sama seperti Bold, menampilkan huruf menjadi lebih tebal.
      8 Emphasis Tag ini memiliki funngi yang sama seperti Italic, menampilkan huruf menjadi miring.
      9 Underline Tag ini menampilan adanya garis dibawah huruf.
      10 SubScript Tag ini menapilkan konten kecil dibawah, CO2.
      11 SuperScript Tag ini menapilkan pangkat pada konten, 15.0002
      12 Preformatted Tag ini menampilkan teks apa adanya pada halaman browser.
      13 Anchor Tag ini mengarahkan ke halaman atau web lain dengan mengkliknya.
      14 Comment Tag ini berfungsi memberikan catatan pada coding yang nantinya tidak akan ikut nampak pada halaman browser.
      15 List Tag ini berfungsi membuat list konten.
      16 Img Tag ini berfungsi untuk menampilkan gambar.
      17 Multimedia Tag ini berfungsi untuk menampilkan medi audio dan video.
      18 Table Tag ini berfungsi untuk menampilkan tabel.
      19 Form Tag ini berfungsi untuk menampilkan form penginputan.
      20 Iframe Tag ini untuk menampilkan frame web youtube atau pdf
